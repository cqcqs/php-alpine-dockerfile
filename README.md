# docker-alpine-php
基于 `Alpine3` 的 `PHP7.4` 镜像

## 目录

- **alpine** 基于 `alpine`
- **php-alpine** 基于 `php:7.4-fpm-alpine`

## 使用

```bash
docker build -t php74-fpm:1.0-alpine .
```

**推荐使用阿里云镜像：**

```bash
docker pull registry.cn-hangzhou.aliyuncs.com/cqcqs/php74-fpm:1.0-alpine
```
